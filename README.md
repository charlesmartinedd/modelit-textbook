### Format Specifications

- **Format:** PDF (version 1.4)
- **Body Text:** 14pt Georgia/Palatino serif
- **Headers:** 42pt H1, 28pt H2, 20pt H3
- **Color Scheme:** ModelIt brand colors (#0F6ACE primary blue, #48d2fc cyan)
- **Images:** Base64-embedded for reliable display
- **File Size:** ~9 MB (includes embedded images)

### Production Pipeline

1. **Content Generation:** AI-generated educational content (OpenRouter)
2. **Illustration:** AI-generated illustrations (Gemini 2.5 Flash Image - "Nano Banana")
3. **Formatting:** Professional HTML/CSS formatting
4. **Rendering:** Playwright PDF rendering with embedded images
5. **Quality Assurance:** GPT-4 Vision quality verification

### Build Requirements

- Node.js (for Playwright PDF rendering)
- Playwright browser automation
- HTML/CSS templates
